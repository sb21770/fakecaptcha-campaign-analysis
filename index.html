Executive Summary

This analysis documents a sophisticated multi-stage fileless malware campaign discovered while attempting to access a payment page on an Israeli volunteer organization website that provides assistance services to people in need.

Upon navigating to the payment page to donate for services, a suspicious CAPTCHA verification popup appeared, masquerading as a Cloudflare security check. The CAPTCHA instructed users to press Win+R, then Ctrl+V, then Enter ‚Äî a classic social engineering technique to execute a malicious PowerShell command that had been silently copied to the clipboard.

Recognizing the suspicious nature of this "verification," the command was not executed. Instead, a full malware analysis was conducted in an isolated environment, revealing a multi-stage info stealer operation using Donut shellcode for fileless execution and Telegram for data exfiltration.

VERDICT: .NET INFO STEALER WITH TELEGRAM EXFILTRATION

Conclusions & Recommendations

Key Findings

  • Active campaign targeting charity/volunteer organization payment pages
  • Uses fake CAPTCHA with Cloudflare branding for social engineering
  • Sophisticated 5-stage fileless attack using Donut shellcode
  • Final payload verified as .NET info stealer (Marte family) via k__BackingField patterns
  • Targets browsers (Chrome, Firefox, Edge, Opera), crypto wallets (Atomic, Guarda), and messaging apps (Discord, Telegram)
  • Exfiltrates stolen data via Telegram Bot API (SendMessage, SendDocument)
  • Multi-architecture support (CommandX64, CommandX86) ‚Äî works on both 32-bit and 64-bit systems

Recommendations
